Robot Poeta Sentimental

Cada paraula respira i es transforma en poesia

Estat:

🌬️ Escoltant el teu cor…

Les teves paraules:

Aquí apareixerà el que diguis...

Anàlisi de sentiments:

Esperant les teves paraules...

Poema generat:

El poeta robòtic respira
esperant les teves paraules
per transformar emocions
en versos que viuen.
— Robot Poeta
